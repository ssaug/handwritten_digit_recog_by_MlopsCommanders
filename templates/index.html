Handwritten Digit Recognition Using Ensemble Method

By MlopsCommanders

This project provides a REST API for recognizing handwritten digits using an ensemble of machine learning models.

How to Use the API

1. Get Model Parameters

Use this endpoint to retrieve the parameters of the trained models.


GET http://localhost:5000/best_model_parameters

2. Predict a Handwritten Digit

Send an image's feature vector (28x28 normalized pixel values) to predict the digit.


POST http://localhost:5000/predict
Content-Type: application/json

{
    "features": [[0.0, 0.1, 0.2, ..., 0.9, 1.0, 0.0]]
}

3. Train the Model

Send new training data to retrain the ensemble models.


POST http://localhost:5000/train
Content-Type: application/json

{
    "features": [
        [0.1, 0.2, ..., 0.9],
        [0.5, 0.6, ..., 0.3]
    ],
    "labels": [5, 3]
}

Experiment Results

Example Input for Prediction API Call:


{
    "features": [
        [0.00, 0.00, 0.33, 0.73, 0.62, ..., 1.00, 1.00, 0.47]
    ]
}

Response:


{
    "predictions": [7]
}

{
    "features": [

        [1.00, 1.00, 1.00, 1.00, 1.00, ..., 1.00, 1.00, 1.00]
